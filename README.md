# Multi-Label Classification of Microcytic Anemia–Related Diagnoses Using Reduced Laboratory Parameters

Microcytic anemia is commonly associated with several underlying conditions, including iron deficiency anemia (IDA), anemia of chronic disease (ACD) and thalassemia. Although these disorders share similar hematological characteristics, their clinical management and treatment strategies differ substantially.

In hospitals, the diagnostic process involves multiple laboratory tests, which can provide overlapping information. Not only is it expensive for laboratories to run but it also affects to patients since they have to bear the additional costs. 

By leveraging available laboratory data, it is possible to identify a subset of key diagnostic indicators that can effectively distinguish between different causes of microcytic anemia. Optimizing the selection of laboratory parameters may help maintain diagnostic accuracy while reducing the number of required tests. This approach can improve diagnostic efficiency, minimize patient burden, and support more cost-effective clinical decision-making.

The dataset is provided by the hospital, therefore it is privately used. No analysis of this dataset is uploaded in this repository.
